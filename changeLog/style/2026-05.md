# Style ChangeLog

## 2026-05-13

- 迁入文风分析接口与文风画像展示逻辑。
- 保留基于参考文本或默认题材生成抽象写作规则的能力。
- 调整文风生成页宽度与素材图谱保持一致，并优化分章生成、审计修订和自动审计模块的列宽与换行，避免内容被挤压裁切。
- 创作灵感/文风画像悬浮面板支持拖动并记忆位置，默认位于返回书籍列表下方；素材图谱内容不再自动混入，改为通过“素材带入/素材更新”显式刷新，且不会带入灵感卡片。
- 修复创作灵感/文风画像悬浮按钮收起态宽度不足导致“素材带入/素材更新”和展开入口换行的问题。
- 调整分章生成与自动审计区域的高度策略：移除固定高度裁切，恢复页面纵向滚动；将目标、冲突、必发、场景、钩子等大纲细节移动到正文版本区上方。
- 重新校准分章生成/审计修订模块：组件高度按浏览器视口贴合到底部，正文框内部滚动；顶部信息区移除大纲描述，改为展示目标、冲突、必发、场景、钩子等执行信息。
- 修复分章生成/审计修订执行信息多值错列导致文本竖排的问题，并约束左右栏高度，确保正文区域显示完整且可独立滚动。
- 将分章生成与自动审计面板统一为等高网格布局：左侧章节侧栏、正文区和右侧审计问题列表都改为各自独立滚动，并让 Step 3 与 Step 4 底部对齐。
- 调整文风生成顶部工具条：首次进入页面时默认收起创作灵感/文风画像浮层，并将 Step 2、标题状态与四个操作按钮整理为同一顶部行的左右对齐布局。
- 调整文风生成页滚动策略：顶部返回/切换/文风生成准备区域改为 sticky，下方分章生成与自动审计区域统一为一屏高；同时在章节数达到目标后停止大纲后台轮询并隐藏“继续加载”提示。

## 2026-05-15

- 新增创作设置页中的 API Key 配置与文风模版管理入口，文风模版改为书籍封面式卡片展示，点击卡片可弹窗查看和编辑完整内容。
- 文风参考编辑弹窗新增文风模版下拉选择；当本地已配置模版时，可直接带入模版内容作为参考文本。
- 优化长参考文本的文风采样策略：由单纯截取开头改为按开篇、中段、后段三段抽样，降低整本参考小说只读前文带来的偏差。
- 新增“整本小说提炼”工作流：在设置页上传整本参考小说后，系统会按章节逐章归纳文风特征，并汇总生成精华版文风模版。
