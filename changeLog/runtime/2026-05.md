# Runtime ChangeLog

## 2026-05-13

- 新建独立 `apps/web` 与 `apps/api` 目录结构。
- 新增脚本 `scripts/check.sh`。
- 新增独立 `.cursorrules`、项目级 skills 和本地配置约束。
- 参考旧项目 `start.sh` 优化本项目启动流程：新增根目录 `start.sh`，启动前检查前后端依赖、配置文件与端口占用，并将问题输出到本地日志；后端默认端口改为 `19081`，启动时按顺序等待后端和前端健康检查通过后再输出访问地址。
- 启动脚本增加端口兜底：默认端口被占用时自动选择冷门备用端口，并输出最终前后端访问地址。
- 启动脚本会将当前实际前后端地址写入 `.tmp/current-dev.env`，避免浏览器继续访问旧备用端口导致 Chrome 错误页。
- 收敛启动入口：删除 `scripts/dev-all.sh`、`scripts/dev-api.sh`、`scripts/dev-web.sh`，以后只使用根目录 `./start.sh` 启动前后端。
